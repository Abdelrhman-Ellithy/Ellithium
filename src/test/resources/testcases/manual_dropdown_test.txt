Title: Verify Dropdown Option Selection
Description: The user navigates to the Dropdown page and selects Option 1, then verifies it is selected.

Preconditions:
1. The user is on the Dropdown page of the HerokuApp internet website.

Steps:
1. Locate the dropdown menu element on the page.
2. Select the option with the visible text "Option 1" from the dropdown.
3. Retrieve the currently selected option from the dropdown menu.

Expected Results:
1. The retrieved selected text should exactly match "Option 1".
